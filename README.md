## 待初始化

<p align="center">
  <img src="https://img.shields.io/badge/状态-待初始化-lightgrey?style=flat-square" alt="待初始化">
</p>

这是一个刚从模板创建的字幕仓库，尚未完成信息初始化。

**下一步：**

1. 进入本仓库的 **[Actions](../../actions)** 标签页
2. 选择 **"初始化字幕仓库 (Initialize Subtitle Repo)"** 工作流
3. 点击 **Run workflow** 手动触发，无需填写任何参数

工作流会自动根据仓库名（`英文片名_年份` 格式）在 TMDB / 豆瓣检索影片信息，并重写本 README 为正式的项目主页。

> [!NOTE]
> 如果仓库名格式不正确，或未能在 TMDB 中匹配到对应影片，工作流会中止并将本 README 替换为对应的错误说明，请届时按提示删除并重新创建仓库。

## Awaiting Initialization

<p align="center">
  <img src="https://img.shields.io/badge/status-awaiting_initialization-lightgrey?style=flat-square" alt="Awaiting Initialization">
</p>

This is a subtitle repository just created from the template and has not yet been initialized.

**Next steps:**

1. Go to this repository's **[Actions](../../actions)** tab
2. Select the **"初始化字幕仓库 (Initialize Subtitle Repo)"** workflow
3. Click **Run workflow** to trigger it manually — no input required

The workflow will look up the movie on TMDB / Douban based on the repository name (in `EnglishTitle_Year` format), then rewrite this README into the project's public homepage.

> [!NOTE]
> If the repository name format is invalid, or no matching title is found on TMDB, the workflow will abort and replace this README with the corresponding error message. Please delete and recreate the repository as instructed.
